# Multimessenger Astronomy Research

This repository documents my journey as an undergraduate ECE student exploring **multimessenger astronomy**, with a focus on real-time detection systems for transients like core-collapse supernovae and black hole natal kicks.

**Long-term Goal**: Develop AI + signal-processing techniques for automated alerts and follow-up observations using systems like SNEWS.

## Current Focus 
- Reading review papers on SNEWS, natal kicks, and multimessenger alerts
- Learning signal processing and machine learning basics
- Understanding gravitational wave data (LIGO basics)
- Building research ideas and simulation prototypes
