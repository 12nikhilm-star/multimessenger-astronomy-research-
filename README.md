# Multimessenger Astronomy Research

## About

This repository documents my journey as an undergraduate ECE student exploring multimessenger astronomy.

My long-term goal is to develop AI and signal-processing techniques for detecting and analyzing transient astrophysical events, including:

- Core-collapse supernovae
- Black hole formation
- Natal kicks
- Gravitational waves
- X-ray and radio observations
- Multimodal AI for astronomy
